# 🛡️ ModerationBot — Профессиональный Discord-бот модерации

Полнофункциональный бот для модерации нескольких серверов с PostgreSQL, интерактивными меню, кнопками и слеш-командами.

---

## 📋 Возможности

| Система | Описание |
|---|---|
| 🛡️ Модерация | Бан, кик, мут (тайм-аут), варн, разбан, размут, purge |
| 🤖 Автомодерация | Антиспам, фильтр ссылок, капс-фильтр, запрещённые слова, лимит упоминаний |
| 🎫 Тикеты | Создание, закрытие, транскрипты, добавление участников |
| 📋 Логирование | Удаления, редактирования, входы/выходы, роли, голосовые каналы |
| 📊 Статистика | Топ модераторов, статистика сервера, история действий |
| ⚠️ Варн-система | Автодействия за N варнов, снятие варнов, апелляции |
| ⚙️ Настройки | Всё настраивается через Discord — кнопки, селекторы, модальные окна |
| 🗄️ База данных | SQLite — файл `data/modbot.db`, без отдельного сервера |
| 🐳 Docker | Деплой одной командой `docker compose up -d` |

## 🚀 Быстрый старт

### Вариант 1: Через Docker (рекомендуется)

```bash
# 1. Клонируйте репозиторий
git clone <ваш-репозиторий>
cd discord_bot

# 2. Создайте .env файл
cp .env.example .env
# Откройте .env и заполните DISCORD_TOKEN и OWNER_IDS

# 3. Запустите бота (SQLite-файл создастся автоматически)
docker compose up -d

# 4. Логи
docker logs -f modbot_bot
```

### Вариант 2: Локально

```bash
# 1. Установите зависимости (PostgreSQL не нужен!)
pip install -r requirements.txt

# 2. Создайте .env
cp .env.example .env
# Заполните DISCORD_TOKEN и OWNER_IDS

# 3. Запустите — БД создастся сама в data/modbot.db
python bot.py
```

## ⚙️ Первичная настройка на сервере

1. Пригласите бота с правами **Administrator**
2. Введите `/setup` — откроется мастер настройки
3. Выберите: создать каналы автоматически или выбрать существующие
4. То же для ролей
5. Для тонкой настройки используйте `/settings`

## 📖 Команды

### Модерация
| Команда | Описание |
|---|---|
| `/ban <участник> [причина] [delete_days]` | Забанить пользователя |
| `/unban <id> [причина]` | Разбанить по ID |
| `/kick <участник> [причина]` | Кикнуть |
| `/mute <участник> <время> [причина]` | Мут через timeout (10m, 2h, 1d, 28d) |
| `/unmute <участник> [причина]` | Снять мут |
| `/warn <участник> [причина]` | Выдать предупреждение |
| `/warnings <участник>` | Посмотреть варны |
| `/clearwarns <участник>` | Снять все варны |
| `/history <участник>` | История наказаний |
| `/purge <кол-во> [участник]` | Удалить сообщения |

### Тикеты
| Команда | Описание |
|---|---|
| `/ticket-setup [канал]` | Разместить панель тикетов |
| `/ticket-add <участник>` | Добавить в тикет |
| `/ticket-remove <участник>` | Убрать из тикета |

### Статистика
| Команда | Описание |
|---|---|
| `/modstats [модератор]` | Статистика модератора |
| `/topmod` | Топ 10 модераторов |
| `/serverstats` | Общая статистика сервера |

### Кейсы
| Команда | Описание |
|---|---|
| `/case <id>` | Просмотр кейса |
| `/reason <id> <причина>` | Изменить причину кейса |
| `/appeal <id>` | Подать апелляцию |
| `/myhistory` | Свои наказания |

### Настройки (только администратор)
| Команда | Описание |
|---|---|
| `/setup` | Мастер первичной настройки |
| `/settings` | Панель всех настроек |

## 🗄️ База данных (SQLite)

Файл создаётся автоматически: `data/modbot.db`. Никакой установки не требуется.

```
guild_config        — конфигурация каждого сервера
mod_actions         — история всех действий модерации
warnings            — предупреждения (варны)
tickets             — тикеты поддержки
ticket_messages     — сообщения тикетов (транскрипты)
mod_stats           — статистика модераторов
banned_words        — запрещённые слова
allowed_links       — белый список ссылок
automod_ignore      — игнорируемые каналы
```

## 🔧 Переменные окружения

| Переменная | Описание | Пример |
|---|---|---|
| `DISCORD_TOKEN` | Токен бота с Discord Portal | `MTM...` |
| `OWNER_IDS` | ID владельцев (через запятую) | `123456789,987654321` |

> **База данных** настройки не требует — SQLite-файл `data/modbot.db` создаётся автоматически при первом запуске.

## 📜 Формат времени мута

| Суффикс | Значение |
|---|---|
| `s` | Секунды |
| `m` | Минуты |
| `h` | Часы |
| `d` | Дни |
| `w` | Недели |

**Примеры:** `30s`, `10m`, `2h`, `1d`, `1w`, `2h30m`  
**Максимум:** `28d` (ограничение Discord)

## ⚠️ Требования к боту (Discord Portal)

При создании бота включите все **Privileged Gateway Intents**:
- ✅ Server Members Intent
- ✅ Message Content Intent
- ✅ Presence Intent

Права бота при добавлении на сервер: **Administrator** (или как минимум: Ban Members, Kick Members, Moderate Members, Manage Messages, Manage Channels, Manage Roles, View Audit Log)
